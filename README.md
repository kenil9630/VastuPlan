# VastuAlign - Professional Vastu Floor Plan Analyzer

VastuAlign is a high-precision interactive tool built for professional interior designers and Vastu consultants. It allows you to overlay an accurate 16-zone Vastu grid on any floor plan image with ease.

## Key Features
- **Premium Interface**: A modern dark-slate aesthetic with advanced glassmorphism.
- **Flexible Calibration**: Orient your map using any degree relative to North (Clockwise or Anticlockwise).
- **Angular Precision Grid**: An interactive helper grid with controllable intervals (e.g., every 2°).
- **Intelligent Snapping**: Automatically align orientation markers to the nearest grid ray for mathematical perfection.
- **Professional Export**: High-resolution PNG output for professional client reports.

## How to use
1. Upload your floor plan image.
2. Click to set the **Center (Brahmasthan)**.
3. Use the sidebar to set your reference degree (e.g., 31°).
4. Click on the map to set the orientation point (supports snapping).
5. Analyze the 16 zones and export the result.
